PHP Overview

  • PHP is a popular programming language-reference
  • Statics say its used by 80% of all websites
  • Its the language that powers WordPress, which is widely used for content management system for website
  • PHP also powers a lot of different frameworks that make web development easier like Laravel
  • PHP is a programming language that many devs use to create Web Applications among other things
  • You can use PHP to add a little interactivity to an HTML page
  • You can use it as a web application engine that creates HTML pages dynamically and sends them over to the browser
  • PHP is often called a scripting language and its an interpreted language
  • You dont need to compile PHP program to run in
  • The PHP interpretor is responsible for interpreting the instructions written in a PHP program when its executed
  • PHP is a dynamically typed language.
  • PHP variables are checked at runtime rather than before the code is executed
  • You do not need to declare types of PHP variables
  • Being interpreted and loosely dynamically typed will make bugs harder to find
  • PHP is written internally in C
  • In its nature PHP is similar to JavaScript
  • PHP supports object-oriented programming and also functional programming

How to setup

  • There are many ways to install PHP on your local machine
  • Best way to install PHP locally is mamp
  • Mamp is a tool that's freely available for all the operating systems
  • PHP is run by a HTTP server, which is responsible for responding to HTTP requests, the ones made by the browser
  • The server is typically Apache or NGINX
  • Then you need a database like MySql
  • MAMP is a packages that provides all of that, and gives you a nice interface to start/stop everything at once
