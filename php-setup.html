PHP Setup

  • There are many ways to install PHP on your local machine
  • The most convient way to install PHP is to use MAMP
  • MAMP is atool that is freely available for all the Operating Systems - Mac, Windows and Linux
  • MAMP is a package that gives you all the tool you need to get up and running
  • PHP is run by a HTTP server which is responsible for responding to HTTP requests, the ones made by the browser
  • You can access a URL with your browser, Chrome, Firefox or Safari and the HTTP server respons with some HTML content
  • A server is typically Apache or NGINX
  • A database is needed, like MySql
  • MAMP is a packages that provides all of that and more and gives you a nice interface to start/stop everything at once
