PHP Strings

  • Strings can be defined with single quotes
  • Strings can be defined with double quotes
  • The big difference between using single and double quotes is that double quotes we can expand variables
  • With double quotes we can use escape characters such as \n (new lines) \t (tabs)
  • We can concatenate two strings using the . operator
  • We can check the length of a stri+ng using the strlen() function
  • We can replace a portion of a string using str_replace()
  • There are many built in function you can work with strings
    • trim(): strips white space at the beginning and end of a string
    • strtoupper(): makes a string uppercase
    • strtolower(): makes a string lowercase
    • ucfirst(): makes the first character uppercase
    • strpos(): finds the first occurance of a substring in the string
    • explode(): to split a string into an array
    • implode(): to join array elements in a string
