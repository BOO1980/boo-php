PHP Numbers

  • Here are function we can use with numbers
    • round(): to round a decimal number, up/down depending if the value is >0.5 or smaller
    • ceil(): to round a decimal number up
    • floor(): to round a decimal number down
    • rand(): generate a random integer
    • min(): find the lowest number in the number passed as arguments
    • max(): find the highest number in the numbers passed as arguments
    • in_nan(): returns true if the number is not a number
