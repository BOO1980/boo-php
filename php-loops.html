PHP Loops

While loops

  • A while loop is the simplest of loops, the loop keeps iterating while condition evauates to true
  • Do while is a similar, but slightly different in how the first iteration is performed
  • In the do while loop, first we do the first iteration and then we check the condition
  • You can use a for loop easily to iterate over an array
  • The for loop is similar to while, but instead of defining the variable used in the conditional before the loop and instead of incrementing the index variable manually its all in the first line
