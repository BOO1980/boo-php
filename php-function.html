PHP Functions

  • A function is composed of an identifier followed by parentheses
  • Inside those parentheses we pass one of more arguments to the function
  • The function does something and when its done it can return a value
  • If there is no value returned the function returns a null
  • Functionss are one of the most important concepts in programming
  • You can use functions to group together multiple instructions or multiple lines of code and give them a name
  • You can call a function anywhere
  • You can also pass arguments to a function, these arguments are called parameters inside the function definition and we call them arguments when we call them in the function
  • You can set the type of parameters of a function
  • You can also set a default value for the parameters
  • A function can return a value
  • Only one value can be returned from a function, not more than one
  • You do this by using the return keyword
  • If the return word is omitted the function returns null
  • The returned value is useful as it tell you the result of the work done in the function, and lets you use its results after calling it
  • You can optionally set the return type of a functuon
  • When you define a varuable inside a function, that variable is local to that function. Meaning its not visible from the outside. When the function ends it just stops existing
  • Varaiable defined outside of the function are not accessible inside the function, this enforces good programming practice, as we can be sure that the function does not modify excternal variables and cause "side effects"
