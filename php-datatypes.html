PHP Data Types

  • Bool: Boolean values (true/false)
  • int: integer numbers (no decimals)
  • float: floating-point numbers (decimals)
  • string: strings
  • array: array
  • object: objects
  • null: a value that means "no value assigned"
  • To get a data value type from a varable we use the var_dump() built in function
  • the var_dump functuon is one of the essential tools in your php debug code
