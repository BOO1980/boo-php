PHP Arrays

  • Arrays are lists of values grouped under a common name
  • You can define an empty array in two different ways
    • $list = []
    • $list = array()
  • An array can be intialised with values
  • Arrays can hold values of any type
  • Arrays can hold other arrays
  • You can access the element in an array using an index number that references and element $anArray[0]
  • Once an array is created, you can append values to it
  • You can use array_unshift() to add the item at the beginning of the array instead
  • Count how many items are in an array using the built in count() funcion
  • You can check if an array contains an item by uisng the in_array() built in function
  • You can also use array_search to get the index
  • Useful functions
    • is_array(): to check if a variable is an array
    • array_unique(): to remove duplicate values from an array
    • array_search(): to search a value in the array and return the key
    • array_reverse(): to reverse and array
    • array_reduce(): to reduce array to a single value using a callback function
    • array_map(): to apply a callback function to each item in the array: typically used to create new array by modifying the values of an existing array, without altering it
    • array_filter(): to filter an array to a sinlge value using a callback function
    • max(): to get the maximum value contained in the array
    • min(): to get the minimum value contained in the array
    • array_count_values(): to count all the values in the array
    • implode(): to turn an array into a string
    • array_pop(): to remove the last item of the array and returns its value
    • array_shift(): same as array_pop() but removes the first item instead of the last
    • sort(): to sort an array
    • rsort(): to sort an array in reverse order
    • array_walk(): similary to array_map() does something for every item in the aaray, but in addition it can change values in the existing array
